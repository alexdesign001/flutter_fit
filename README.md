# flutter_fit

一个运动手环应用（UI仿华为健康）。只是有UI，没有和手环设备交互功能（不知道华为手环BLE协议,功能是无法做的）。
做此项目目的：初入flutter，熟悉各种Widget属性。

刚拼了几个界面，整体很像之前做项目里用到 QT的QML 和 React+TypeScript
如果只是对于写界面来说的话，个人感觉比写原生的xml要舒服很多
